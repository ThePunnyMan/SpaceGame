Development Progress on UnNamed Space Game

Thes wesite is to provide updates on group 7's progress on their upcoming game. Updates are posted each saturday about the progress made during the week

    Progress:

  • September 22, 2018 - We created a unity project and activated the collaboration option so all group members could work together. We created a prototype to test basic player controls. The "player character" can move with wasd and a cursor is generated to show where the player is aiming. The player can shoot by pressing the mouse button or holding the button down. A restriction has been placed on the weapons fire rate. There still isnt any collision detection.
  • Spetember 15, 2018 - Met to discuss overall design of the game to prepare for the game proposal presentation. Downloaded Unity to start developement.
